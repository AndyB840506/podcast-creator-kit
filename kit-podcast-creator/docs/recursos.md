# Recursos para Podcasters

Referencia completa de herramientas, plataformas y servicios para producir tu podcast. Organizado en 7 secciones.

---

## 1. Música sin derechos de autor

Para intros, outros, stingers y música de fondo.

| Plataforma | Costo | Licencia | Mejor para | Notas |
|---|---|---|---|---|
| **Epidemic Sound** | ~$15/mes | Licencia comercial incluida | Uso profesional, muchas opciones | La más completa, amplio catálogo |
| **Artlist** | ~$200/año | Licencia comercial incluida | YouTube, podcasts, videos | Ilimitado, descuentos anuales |
| **Free Music Archive** | Gratis | Creative Commons | Investigación, catálogo pequeño pero útil | fma.bandcamp.com |
| **YouTube Audio Library** | Gratis | Licencia especial (TOS) | Calidad alta, sin crédito requerido | studio.youtube.com/channel/music |
| **Incompetech** | Gratis | CC0 (requiere crédito) | Géneros variados, música original | incompetech.com — solo menciona autor |
| **Pixabay Music** | Gratis | Pixabay License | Descarga rápida, sin registro | pixabay.com/music — sin crédito |
| **ccMixter** | Gratis | Creative Commons | Comunidad de músicos, remixes | ccmixter.org — comunidad activa |

**Para empezar:** YouTube Audio Library (gratis, alta calidad, sin requisitos de crédito).

**Necesitas generar/descargar:**
- **Intro** (5-15 seg) — tema que se repite al inicio de cada episodio
- **Outro** (10-30 seg) — tema que se repite al cierre
- **Stinger** (2-3 seg) — sonido para transiciones internas
- **Música de fondo** (loop suave) — opcional, para segmentos específicos

## 2. Software de grabación y edición

### Grabación

| Software | Plataforma | Costo | Para quién | Curva aprendizaje |
|---|---|---|---|---|
| **Audacity** | Windows / Mac / Linux | Gratis | Empezar. Interface clara. | Baja |
| **GarageBand** | Mac (incluido en OS) | Gratis | Mac: más intuitivo que Audacity | Baja |
| **Descript** | Windows / Mac | Freemium / $12/mes | Edita audio como documento de texto. Transcripción automática. | Media |
| **Adobe Audition** | Windows / Mac | $20/mes (Creative Cloud) | Profesional, muchas herramientas. | Alta |
| **Hindenburg Journalist** | Windows / Mac | $95 USD (una sola vez) | Especialmente para podcasters. Excelente para entrevistas. | Media |

**Recomendación:** 
- Windows/Linux: **Audacity** (gratis)
- Mac: **GarageBand** (gratis, incluido)
- Si quieres más control: **Descript** (freemium, muy intuitivo)

## 3. Grabación remota (para entrevistas)

| Plataforma | Costo | Características | Mejor para |
|---|---|---|---|
| **Riverside.fm** | Freemium / $15/mes | Pistas separadas, grabación en alta calidad, interfaz clara | Entrevistas de calidad profesional |
| **Zencastr** | Freemium / $18/mes | Similar a Riverside, muy estable | Alternativa popular a Riverside |
| **SquadCast** | Freemium / $12/mes | Más económica, buena estabilidad | Presupuesto limitado |

**Recomendación:** Empieza con Riverside.fm (freemium es suficiente).

**Especificación importante:** Elige plataformas que graben pistas separadas por participante — así editas cada voz por separado después.

## 5. Directorios de podcasts

Una vez tienes RSS, registra tu podcast en:

- **Spotify** (creators.spotify.com) — automático desde Spotify for Creators
- **Apple Podcasts** (podcastsconnect.apple.com) — requiere Apple ID, aprobación 1-5 días
- **Amazon Music / Audible** (music.amazon.com/podcasts/submit)
- **iVoox** (ivoox.com) — especialmente si audiencia hispanohablante
- **Pocket Casts, Overcast** — se alimentan del RSS automáticamente

**Nota:** Cada plataforma maneja su propio directorio. El RSS se propaga automáticamente a muchas si estás en Spotify o Buzzsprout.

## 6. Herramientas de transcripción

Después de grabar, transcribir te ayuda a: (1) crear show notes, (2) mejorar SEO, (3) extraer quotes para redes.

| Herramienta | Costo | Precisión | Características | Mejor para |
|---|---|---|---|---|
| **Whisper (OpenAI)** | Gratis | Excelente | Open source, descarga local, sin límite | Control total, privacidad |
| **TurboScribe** | Freemium / $10/mes | Muy buena | Interfaz fácil, transcripción rápida | Uso general, interfaz bonita |
| **Descript** | Incluido si usas editor | Muy buena | Integrado con edición, ya tienes el audio | Si ya usas Descript |
| **Otter.ai** | Freemium / $17/mes | Bueno | Colaborativo, puedes compartir con invitado | Entrevistas con anotaciones en tiempo real |

**Recomendación:** **TurboScribe** (interfaz amigable) o **Descript** (si ya lo usas para edición).

## 7. Plataformas de IA para imágenes

Para generar portadas de episodios y el artwork del podcast.

### Tabla comparativa

| Plataforma | Costo | Nivel | Mejor para | Notas |
|---|---|---|---|---|
| **Google Flow / Imagen 3** | Gratis (con cuenta Google) | Excelente | Retratos fotorrealistas, portadas limpias | Recomendado para empezar |
| **Kling AI** | Freemium / ~$10/mes | Muy bueno | Composiciones cinematográficas, videos | klingai.com |
| **Midjourney** | ~$10/mes básico | El más artístico | Estilos consistentes, reutilizables | Requiere Discord, comunidad fuerte |
| **Adobe Firefly** | Incluido en Creative Cloud / ~$5/mes | Muy bueno | Integración con Photoshop, uso comercial seguro | Si ya tienes Creative Cloud |
| **DALL-E 3** | Incluido en ChatGPT Plus $20/mes | Excelente | Sigue instrucciones detalladas de texto | Si ya tienes ChatGPT Plus |
| **Stable Diffusion** | Gratis (instalación local) | Máximo control | Control total, máxima privacidad | Curva de aprendizaje alta |

### Especificaciones técnicas para Spotify

Portada del episodio (formato principal):
- **Dimensiones:** 3000 × 3000 px (cuadrado exacto)
- **Formato:** JPEG o PNG
- **Tamaño máximo:** 500 KB
- **Color:** RGB (no CMYK)
- **Resolución:** 72 DPI mínimo

**Importante:** Los modelos de IA generan PNG de alta resolución (100+ MB).

**Antes de subir a Spotify:**
1. Convierte a JPEG
2. Comprime con [squoosh.app](https://squoosh.app/) (gratis)
3. Verifica que quede bajo 500 KB

### Costos y límites

> **⚠️ Advertencia:** Los planes gratuitos tienen límite de créditos/tokens por mes. Una sesión con varias iteraciones puede consumir rápido los créditos.
>
> **Para uso constante (1 episodio/semana):**
> - Freemium + iteraciones cuidadosas: posible
> - Suscripción mensual: recomendada si generas 2+ imágenes/semana

### Recomendación para empezar

**Google Flow es gratis y produce resultados excelentes.** El kit genera prompts optimizados para este modelo. Cuando el podcast crezca, puedes evaluar otras opciones.

## Matriz de costo total (primer año)

**Setup mínimo (gratis):**
- Hosting: Spotify for Creators (gratis)
- Grabación: Audacity/GarageBand (gratis)
- Música: YouTube Audio Library (gratis)
- Imágenes: Google Flow (gratis)
- **Total: $0/mes**

**Setup recomendado:**
- Hosting: Spotify for Creators (gratis)
- Grabación: Descript ($12/mes)
- Música: Incompetech o YouTube Audio Library (gratis)
- Imágenes: Google Flow (gratis)
- Entrevistas remotas: Riverside.fm freemium (gratis)
- Transcripción: TurboScribe ($10/mes)
- **Total: ~$22/mes**

**Setup profesional:**
- Hosting: Transistor ($19/mes)
- Grabación: Adobe Audition + Descript ($20 + $12/mes)
- Música: Epidemic Sound ($15/mes)
- Imágenes: Midjourney ($10/mes)
- Entrevistas remotas: Riverside.fm pro ($15/mes)
- Transcripción: Otter.ai ($17/mes)
- **Total: ~$98/mes**

## Flujo recomendado de herramientas

```
Podcast → Grabación (Audacity/GarageBand)
         ↓
        Edición (Descript o Adobe Audition)
         ↓
        Transcripción (TurboScribe o Whisper)
         ↓
        Artwork (Google Flow)
         ↓
        Hosting (Spotify for Creators)
         ↓
        Distribución (automática a múltiples plataformas)
```

## Checklist de instalación inicial

- [ ] Elige plataforma de hosting (recomendación: Spotify for Creators)
- [ ] Descarga e instala software de grabación (Audacity o GarageBand)
- [ ] Descarga música de intro/outro de YouTube Audio Library o Incompetech
- [ ] Crea cuenta en Google para acceder a Google Flow (imágenes)
- [ ] Configura cuenta de Spotify for Creators
- [ ] Opcional: descarga Descript (o TurboScribe) si quieres transcripción más fácil
- [ ] Opcional: configura Riverside.fm si planeas entrevistas
